# portfolio

## Projects
### Booth's Algorithm
- Implemented Booth’s Algorithm to perform signed 4-bit binary multiplication by simulating hardware-level register operations. 
- The program models the A, Q, and Q-1 registers, applies two’s complement arithmetic, and performs arithmetic right shifts to compute the final product.
- [Github Repository](https://github.com/elister0/Booths-Algorithm)

### IEEE754 Simulator
- Developed a simulator that converts decimal floating-point numbers into their IEEE 754 32-bit binary representation and performs addition or multiplication operations. 
- The program manually computes the sign bit, biased exponent, and 23-bit mantissa by normalizing the input value and constructing the binary representation without using built-in bit-conversion utilities.
- [Github Repository](https://github.com/elister0/IEE7545-Simulator-)

### MIPS Assembly Program
- Completed a series of 14 MIPS assembly language programs using the SPIM simulator to strengthen low-level programming and computer architecture skills.
- Projects implemented control flow, arithmetic operations, memory access, and stack management while simulating register-level computation.
- [Github Repository](https://github.com/elister0/MIPS-Assembly)

### Student Course Registration Simulation
- Designed and implemented a course registration system simulation supporting 900+ students and 30+ courses using custom data structures.
- The system manages student enrollment, course capacity constraints, and registration logic while maintaining data consistency across linked structures.
- [Github Repository](https://github.com/elister0/Student-Course-Registration-Simulation)

### Minesweeper Recreation
- Developed a fully functional Minesweeper game implementing grid-based state management, mine placement logic, and recursive cell-reveal behavior.
- The program dynamically tracks user input, win/loss conditions, and adjacent mine counts while maintaining consistent board state.
- [Github Repository](https://github.com/elister0/Minesweeper)
